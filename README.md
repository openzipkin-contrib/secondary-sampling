## Brave integration
Setup this:
```xml
  <repositories>
    <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
    </repository>
  </repositories>
```

Then depend on this:
```xml
  <dependency>
    <groupId>io.github.openzipkin-contrib.zipkin-secondary-sampling</groupId>
    <artifactId>brave-secondary-sampling</artifactId>
    <version>master-SNAPSHOT</version>
  </dependency>
```
